Atom¶

Shortcuts¶

Action Keys
Open command palette Ctrl-Shift-P
Open settings Ctrl-,
Toggle tree view visibility Ctrl-\
Toggle focus on tree view Alt-\
Reveal active file in tree view Alt-Shift-\
Close current tab Ctrl-W Esc
Open fuzzy finder Ctrl-P
Toggle keybinding resolver panel Ctrl-.
Toggle Developer Tools Ctrl-Shift-I

Command Line¶

  • Export installed packages & themes to ~/dotfiles/home/.atom_installed:

    $ apm list --installed --bare > ~/dotfiles/home/.atom_installed

  • Install packages & themes from ~/.atom_installed:

    $ apm install --packages-file ~/.atom_installed
