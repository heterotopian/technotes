    • Storing command results (output/status) in variables:

      OUTPUT=$(command)
STATUS=$?

      Branch on status:

      if command
then
    echo "success"
fi

if OUTPUT=$(command)
then
    echo "success - ${OUTPUT}"
fi

      Pipelines:

      set -o pipefail
find /invalidpath | wc -l  # Returns last non-zero subcommand status

    • Reading files line-by-line:

      {
    while IFS= read -r LINE
    do
        printf '%s\n' "${LINE}"
    done
    # Edge case: last line isn't newline terminated
    # `read` consumes LINE but returns false
    # Handle leftover partial line
    if [[ -n ${LINE} ]]
    then
        printf '%s\n' "${LINE}"
    fi
}

    • Using ‘find’:

      while IFS= read -r -d '' LINE
do
    printf '%s\n' "${LINE}"
done < <(find -name "$PATTERN" -print0)
